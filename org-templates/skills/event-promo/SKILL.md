## Promotion Formats

### Email Invitation
- Subject line: benefit-focused, include event name
- Body: 3-paragraph max (what, why you, CTA)
- Include: date, location, registration link
- For exclusive events: emphasize limited capacity and peer-caliber environment

### Social Post
- Lead with the value proposition, not the event name
- Include key stats (attendee count, past NPS, partner count if impressive)
- Use event-specific hashtags

### One-Pager / Sell Sheet
- Header: event name + tagline
- Key stats in callout boxes
- Agenda highlights (3-5 bullets)
- Testimonial quote from past attendee (if available)
- Registration CTA with deadline

### Speaker/Sponsor Outreach
- Lead with audience quality and reach
- Include audience demographics (seniority, industry, company size)
- Reference past speakers or sponsors by name if public and relevant
- Be specific about what they get (speaking slot, visibility, data, access)

---

## Process
1. Identify which event and which promotional format
2. Confirm target audience (current members/customers, prospects, sponsors, speakers, media)
3. Draft content with event-specific messaging
4. Include all logistics (date, location, registration link)
5. Present for review — **always confirm before sending or posting**
